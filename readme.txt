1. Распаковать в пустую папку.

2. Открыть терминал в корне проекта и ввести команду: npm i gulp --save-dev

3. Работать в проекте. html писать в index.html, scss в папке scss. Изображения
и шрифты ложить в папку app.

Для отображения результата, терминал запустить с командой gulp

Для сборки готового проекта ввести команду: gulp build
